## Glossary (chapter-local)

| Term | Definition per text |
|------|---------------------|
| Semantic search | Search by meaning via embeddings, not just keywords |
| Dense retrieval | k-NN in embedding space over document/chunk vectors |
| Reranker | Model scoring (query, document) pairs to reorder shortlist |
| RAG | Retrieve relevant texts then generate answer conditioned on them |
| Grounded generation | LLM answer constrained by retrieved context |
| Hybrid search | Combines dense semantic + lexical (e.g. BM25) retrieval |
| MAP | Mean average precision тАФ IR metric over ranked results |
| monoBERT | Cross-encoder BERT reranker per query-document pair |
| Agentic RAG | LLM plans multi-step retrieval across tools/sources |
